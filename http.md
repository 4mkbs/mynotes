http stand for hypertext transfer protocol
foundation of www
http is application layer protocol (layer 7 of osi model)
http request is way for client to communicate with server
http  request carries series of encoded information in the form of headers and body
# http methods
defination: http methods are a set of request methods to indicate the desired action to be performed for a given resource
GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS, TRACE, CONNECT
get: for retrieving data from server
post: for sending data to server
put: for updating existing resource
delete: for deleting existing resource
patch: for partially updating existing resource
head: for retrieving headers only
options: for retrieving supported methods and other options

# http headers
definition: http headers are key-value pairs sent between the client and server to provide additional information about the request or response
common headers: 
- Content-Type
- Content-Length
- User-Agent
- Accept
- Authorization
